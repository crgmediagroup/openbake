OpenBake

OpenBake is, collectively, a few PHP scripts that can be used for baking versions of your SaaS for distribution for clients.

Info

In the source code is a fancy frontend, index.php, and a backend script (bake.php) which relies on zip-folder.php. A directory marked "demosaas" is included which contains a demonstration SaaS that the program can copy. It is not required for overall function of the program.

Limitations

This system has almost ZERO error handling and completely relies on the user knowing what they are doing when fill out the frontend or else. The system also requires readwrite permissions in several places as noted in the scripts, which have a sort of step by step guide in them detailing what the code does.

Use Cases

The system could be used to automatically bake installations of a PHP SaaS, such as inputting a generated random product key in the config file, which it then throws into a database. That same SaaS could use OpenVerify to check if that product key is legitimate against that same database. These two programs are designed to easily integrate with each other.
